# 物品修饰符

修饰符是物品的额外属性，它相对独立存在，不与物品的等级、品质有关联。修饰符的唯一作用就是对物品特定的属性进行修改，负面或正面修饰皆可能存在。修饰符会在物品生成时随机选择，但也有可能完全没有修饰符。

本页将列出物品修饰符的种类、修饰的属性、具体的数值、以及适用的物品类型。
